# Event Management System (OOPD Project)

This project is a console-based Event Management System developed in C++
using Object-Oriented Programming principles. The system is designed to
manage events, participants, and related operations efficiently through
structured class-based design.

------------------------------------------------------------------------

## Project Overview

The Event Management System allows users to:

-   Create and manage events
-   Register participants
-   Store and organize event-related details
-   Demonstrate core OOP concepts through practical implementation

This project highlights structured program design and real-world problem
modeling using C++.

------------------------------------------------------------------------

## Concepts Implemented

The project demonstrates the following Object-Oriented Programming
concepts:

-   Classes and Objects
-   Encapsulation
-   Abstraction
-   Constructors and Destructors
-   Member Functions
-   File Handling 
-   Basic Data Management Logic

## How to Compile and Run

1.  Open terminal in the project directory.

2.  Compile the program:

        g++ final_oopd_project.cpp -o event_system

3.  Run the executable:

        ./event_system
